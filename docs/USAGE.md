# Utilisation

## Raccourcis
- ALT : déplacer l’icône
- SHIFT + clic sur l’icône : options
- /AnV2 icon : afficher/masquer l’icône

## Fonctions principales (UI)
- Affiche vos familiers (normal/rare)
- Affiche la nourriture pour vos familiers

## Données & synchro
- Les données sont enregistrées à chaque connexion/déconnexion.
